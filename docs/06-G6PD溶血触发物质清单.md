# G6PD 缺乏症（蚕豆病）溶血触发物质清单

> 这是一份面向父母的“就医沟通提示清单”，不是处方，也不是完整药物禁忌表。不同 G6PD 变异、剂量、年龄、感染状态和药品制剂会改变风险；任何药物、外用制剂或中成药都应先把“宝宝有/疑似 G6PD 缺乏”告诉医生和药师。

## 家庭环境与食物

| 类别 | 需要重点避免或先咨询 | 说明 |
|---|---|---|
| 食物 | 蚕豆及蚕豆制品 | 可能诱发急性溶血；添加辅食后尤其要注意配料表。 |
| 家庭用品 | 樟脑丸、萘丸及含萘防虫用品 | 萘暴露可导致溶血，G6PD 缺乏者风险更高；不要放入宝宝衣柜、被褥或收纳箱。 |
| 染料/外用 | 海娜粉/指甲花 | 既往病例报告提示可诱发严重溶血；婴幼儿不应使用。 |

## 药物与中成药：必须先核对

下面只列出项目中与新生儿/婴儿相关性较高、需要主动告知医生的候选项，不等于对每个个体的绝对禁用结论：

- 亚甲蓝（甲基蓝）
- 伯氨喹（用于疟疾相关治疗，必须先检测并由专科医生决定）
- 达普松
- 磺胺类药物，包括磺胺甲噁唑等
- 呋喃妥因、萘啶酸等需结合具体适应证和剂量核对的药物
- 氯霉素及含氯霉素的外用/眼用制剂
- 含茵栀黄、黄连、牛黄、金银花等成分的中成药或草药制剂

## 出现这些情况要尽快就医

接触可疑诱因后，如果出现黄疸迅速加深、脸色苍白、尿色明显变深、精神差、吃奶下降、发热、呼吸急促或异常嗜睡，不要在家继续观察或自行“退黄”，应立即就医，并带上药品、食品或用品包装。
