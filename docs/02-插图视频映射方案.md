# 母乳喂养插图与教学视频映射方案

> 这份方案是给电子书和 AI 调取资源用的。插图现在按“正文知识点锚点”挂载，而不是只给每章放一张封面图。

## 映射原则

- 每个真正承担判断、流程、对比或行动建议的知识点，都可以拥有独立插图位。
- 纯资料链接、重复解释、长表格和需要精确阅读的医学原图，不强行生成火炬插画，直接挂原始资料。
- 插图优先做“结构图、流程图、对照图、时间线图”。
- 视频优先做“操作演示、场景说明、误区拆解、医生/顾问讲解”。
- 所有资源命名必须带章号，避免后续混乱。

## 知识点插图挂载规则

正文目录使用稳定锚点 `point-章节号-序号`。如果对应图片存在，构建脚本会自动把它插入该知识点标题之后：

```text
assets/images/article-illustrations/point-01-001.png
assets/images/article-illustrations/point-03-007.png
assets/images/article-illustrations/point-07-011.png
```

这样新增图片不需要手工修改整本 README，也不会让目录锚点和正文位置漂移。每张图必须使用火炬手 IP，纯白背景、手绘线稿、少量批注；WHO 曲线和黄疸阈值等需要精确读数的资料，保留官方图表或原始图片作为资料图。
