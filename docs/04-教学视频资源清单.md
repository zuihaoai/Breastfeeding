# 母乳喂养教学视频资源清单

> 这份清单先收“可外链、可复用、权威度高”的公开视频页面，不直接复制视频文件。第一批以 Stanford Medicine / First Droplets / Global Health Media 为主。

## 使用原则

- 优先链接外部官方页面，不把视频本体复制进仓库。
- 一条视频可以对应多个章节，目录只保留最强匹配项。
- 页面若明确标注商业限制，仓库只保存引用和说明。
- 先补动作类视频，再补概念类视频。
