# 母乳喂养知识库目录总表

> 这是 `docs/母乳喂养/` 的原料层真源目录。后续做电子书、知识卡片、Skill 索引和插图/视频映射时，都以这里的章节编号为锚点。

## 目录使用方式

- 父母阅读：按时间线读，先孕期，再初乳，再产后一周，再 0-6 个月，再进到母乳喂养、辅食、人工喂养。
- 奶爸AI检索：用“文件名 + 章节名 + 关键词”三段式检索。
- 电子书导入：每个文件先保留为一个大章，后续再按小节拆成可单独引用的章节块。

### 02 初乳

- 初乳始于无，关键在吮吸
  - 开奶之前有个常被误解的空转期
  - 泌乳刺激的因果循环
  - 空转期的生理解释
  - 早产、顺产、肌肤相亲与开奶
- 开奶秘诀：早接触、早吸吮、勤吸吮
  - 早接触（热车）
  - 早吸吮（点火）
  - 勤吸吮（踩住油门）
  - 无营养吮吸
  - 给父母们的行动建议
- 初乳很少，一勺就饱：奶少，胃也小
  - 初乳是第一剂疫苗
  - 认识“半天挤不出一茶匙”的正常性
- 干扰初乳的手段：母婴分离；让母婴多休息；怕娃渴饿、脱水、低血糖
  - 母婴分离：新生儿是否需要立即洗澡
  - 国际医学指南的明确推荐
  - 为什么“国内医院一出生就洗澡”
  - 怕娃渴饿、低血糖、脱水
  - 有父母会好奇，水又没有营养，喂水如何干扰母乳喂养
  - 诱导喂糖水
  - 配方奶粉只是一种富含食品添加剂的加工食品
  - 让母婴多休息
  - 常见情况：胎便污染羊水
  - 早产儿和坏死性小肠结肠炎
